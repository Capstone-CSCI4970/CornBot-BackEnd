SamplingMethods.lowestPercentage(ml_model, X_test, n)¶

This sampling method finds the pictures with the lowest percent probability in the test set. It then removes the n samples with the lowest percent probability from X_test. Finaly it returns the new testing set and the sample set.

Parameters

  • al_model (active learning class object) – The active learning model.

  • n (int) – The number of samples to be returned.

Returns

  • X_sample (pandas DataFrame) – The new list of samples to be added to the train set.

  • X_test (pandas DataFrame) – The new testing set with the samples removed.
