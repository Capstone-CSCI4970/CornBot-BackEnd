Viz module¶

Viz.make_confusion_matrix(cf, group_names=None, categories='auto', count=True, percent=True, cbar=True, xyticks=True, xyplotlabels=True, sum_stats=True, figsize=None, cmap='Blues', title=None)¶

This function will make a pretty plot of an sklearn Confusion Matrix cm using a Seaborn heatmap visualization.

Parameters

  • cf (nd-array) – confusion matrix to be passed in

  • group_names (list) – List of strings that represent the labels row by row to be shown in each square.

  • categories (list) – List of strings containing the categories to be displayed on the x,y axis. Default is ‘auto’

  • count (boolean) – If True, show the raw number in the confusion matrix. Default is True.

  • normalize (boolean) – If True, show the proportions for each category. Default is True.

  • cbar (boolean) – If True, show the color bar. The cbar values are based off the values in the confusion matrix. Default is True.

  • xyticks (boolean) – If True, show x and y ticks. Default is True.

  • xyplotlabels (boolean) – If True, show ‘True Label’ and ‘Predicted Label’ on the figure. Default is True.

  • sum_stats (boolean) – If True, display summary statistics below the figure. Default is True.

  • figsize (tuple) – Tuple representing the figure size. Default will be the matplotlib rcParams value.

  • cmap (String) –

    Colormap of the values displayed from matplotlib.pyplot.cm. Default is ‘Blues’

    See http://matplotlib.org/examples/color/colormaps_reference.html

  • title (String) – Title for the heatmap. Default is None.

Returns

confusion_matrix – image uri of confusion matrix also with metrices with score

Return type
